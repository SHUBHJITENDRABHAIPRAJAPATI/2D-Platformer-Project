# 2D-Platformer-Project

A fun and interactive 2D platformer game built with HTML5, CSS, and JavaScript.

## Game Description

This is a classic 2D platformer game where you control a character that needs to jump across platforms to reach the goal flag. Navigate through the level by jumping from platform to platform while avoiding falling off the bottom of the screen.

## How to Play

1. **Start the Game**: Open `index.html` in your web browser
2. **Controls**:
   - **Arrow Keys** or **WASD**: Move left, right, and jump
   - **Left Arrow** or **A**: Move left
   - **Right Arrow** or **D**: Move right
   - **Up Arrow**, **W**, or **Space**: Jump
3. **Objective**: Navigate the platforms and reach the golden flag at the end of the level
4. **Warning**: Don't fall off the bottom of the screen or you'll have to start over!

## Features

- Smooth player movement and jumping mechanics
- Realistic gravity and physics
- Multiple platforms to navigate
- Collision detection
- Win condition (reach the flag)
- Responsive controls (Arrow keys or WASD)
- Visual feedback with a character that has eyes
- Colorful graphics and gradients

## Installation & Running

1. Clone this repository or download the files
2. Open `index.html` in any modern web browser (Chrome, Firefox, Safari, Edge)
3. No additional installation required - it runs directly in your browser!

## Technical Details

- **Canvas Size**: 800x600 pixels
- **Physics**: Custom gravity and collision detection
- **Frame Rate**: 60 FPS (using requestAnimationFrame)
- **Player Size**: 30x30 pixels
- **Platforms**: 8 platforms of varying sizes and positions
